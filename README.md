# Student Performance Analysis

## Project Overview
This project focuses on analyzing student performance data to understand how various factors such as **gender, parental education, lunch type, and test preparation courses** affect students' academic scores.

The analysis aims to extract meaningful insights that can help **educators, parents, and policymakers** improve learning outcomes.

---

## Dataset Description
The dataset contains the following features:

| Column Name | Description |
|------------|-------------|
| gender | Gender of the student |
| race/ethnicity | Studentâ€™s ethnic group |
| parental level of education | Highest education level of parents |
| lunch | Type of lunch (standard / free or reduced) |
| test preparation course | Whether the student completed the test prep course |
| math score | Math exam score |
| reading score | Reading exam score |
| writing score | Writing exam score |
| total score | Sum of math, reading, and writing scores |

---

##  Objectives
- Analyze performance differences based on **gender**
- Study the impact of **parental education**
- Evaluate the effect of **test preparation courses**
- Compare performance across **lunch categories**
- Identify trends and patterns in academic performance

---

## Tools & Technologies Used
 Microsoft Excel

---

##  Key Insights
- Female students have a **higher average score** in reading and writing.
- Students whose parents have **higher education levels** tend to perform better.
- Completing a **test preparation course significantly improves** reading and writing scores.
- Students with **standard lunch** generally score higher, indicating the role of financial support.
- Overall academic performance improves when multiple supportive factors are present.

##  Actionable Recommendations
- Encourage **parental involvement** in students' education.
- Promote **test preparation courses** for students struggling in reading or writing.
- Provide **nutritional and financial support** to students for better performance.
- Focus on early academic intervention for underperforming groups.
